About the Project - 

This is Task-1 for the DecodeLabs Internship. 
The project focuses on Data Cleaning and Analysis using the dataset.
The main goal was to explore the dataset, perform data cleaning, and derive insights that can help in decision making.


 Key Learnings -
 
- Learned how to clean and preprocess real-world e-commerce data using excel
- Improved skills in data visualization to represent order status, product sales, and referral performance
- Understood how coupon codes and referral sources impact sales and customer acquisition

 Key Insights -
 
- Monitors and Tablets are the top-selling products
- Credit Card and Debit Card are the most used payment methods  
- Most orders are "Shipped", with some "Cancelled" and "Returned"
- Instagram and Facebook drive the most customer traffic
